## Overview

Export all tasks from a project board as a CSV file with one click. Useful for reporting, offline backups, or migrating data to other project management tools.

## How to Export

1. Open any project board
2. Click the **Export CSV** button in the toolbar
3. A `.csv` file downloads with all tasks from the project

## Exported Fields

The CSV includes the following columns:

| Column | Description |
|--------|------------|
| Title | Task title |
| Description | Full task description |
| Status | Current status (backlog, in_progress, review, done) |
| Priority | Priority level (urgent, high, medium, low) |
| Created | Creation timestamp |
| Updated | Last update timestamp |

## Notes

- Soft-deleted tasks (in Trash) are **not** included in the export
- The export includes all tasks regardless of status — filter in your spreadsheet tool if needed
- Only project members with access to the board can export
